Следите за ТИЦ и Alexa Rank прямо из консоли WP!

== Description ==

Виджет для консоли, который показывает текущие значения ТИЦ и Alexa Rank вашего сайта.

Замечания:

*   Для работы необходимо наличие cURL на хостинге.

== Installation ==

1. Unzip files.
2. Upload 'wp-seo-status' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
